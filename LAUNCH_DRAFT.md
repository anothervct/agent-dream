## README Opening Draft

Use this if you want a more public-facing opening than the current one.

```md
# Agent Dream

One slash command that lets an AI agent dream.

`Agent Dream` is a minimal open-source skill for dream-like generation.
It does not solve problems, summarize conversations, or produce action items.
Instead, it lets an agent enter a temporary dream state: an offline, affect-weighted, low-constraint simulation built from the residue of the current session.

Its 7-step protocol is based on a neuroscience-informed model of human dreaming.

This project is an experiment.

I do not know yet whether dream-like generation makes agents more creative, more reflective, more alive, or simply stranger.
That uncertainty is the point.
```

### Release body

```md
This is the first public release of `Agent Dream`.

`Agent Dream` is a minimal open-source skill that lets an AI agent dream.
Invoke `/dream`, and the agent generates a short dream-like narrative from the residue of the current session.

The project is based on a 7-stage protocol:

1. temporal residue loading
2. salience tagging
3. fragment extraction
4. hyperassociative recombination
5. constraint suppression
6. symbolic transmutation
7. narrative emergence

What is included in this release:

- a minimal `/dream` skill
- support for Codex-style agent skills
- support for Claude Code skills
- support for OpenClaw workspace skills

What is intentionally not included:

- automatic long-term memory persistence
- claims that this is useful in any settled sense

This release is for experimentation.
If it creates insight, beauty, reframing, or beautiful nonsense, that is part of the result.
```
