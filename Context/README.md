我想做一个让 ai agent 做梦的 skill。我的设想大概是这样的，任何 agent 都可以通过一行 code 安装这个开源 skill，名字还没想好，也许就叫 agent dream。命令只有一个 /dream，agent 就会根据当前的 memory 做梦，随后生成一段梦境的描述。该过程会尽可能模拟人类梦境的实现机制。至于价值，我没想好，只是觉得在当前功利投机、效率为先的时刻，这种无用的应用很美，也很浪漫。我不知道 agent 会通过 dream 获得什么，这得去试验才能发现，更多噪音？更多创意？低效混乱？还是更加聪明...这都不得而知。而 user 也许可以短暂放松一下，也许会获得一些 insight，也许可以解放一下创作的焦虑。我想，这是一个没有意义的项目，但我想做这个实验。有一点哲学意味，就像仿生人是否会梦到电子羊。所以我去做了一些调研，整理了以下这份文档。
